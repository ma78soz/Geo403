# Beispieldaten

Enthalten sind Rasterdaten, um Beispielhaft eine Hangrutschungsgefährdungskarte basierend auf dem Statistical Index zu berechnen. 
- Sichtfeld (Viewshed)
- Hangrutschungen
- 12 Einflussparameter
  - DGM Derivate:
    - Hangneigung
    - Hangausrichtung
    - Planare Krümmung
    - Profilkrümmung
    - SPI
    - TWI 
  - Landbedeckung
  - Lithologie
  - Boden
  - Distanz zu Straßen
  - Distanz zu Gewässern
  - Niederschlag
